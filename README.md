# Resume Generator

A web-based resume generator that allows users to create professional resumes by filling out a form. The application generates a PDF download of the resume.

## Features

- User-friendly form for personal information, education, skills, work experience, and accomplishments
- Profile photo upload
- Certificate/credential attachment support
- PDF export with professional formatting
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- html2pdf.js (for PDF generation)
- PDF.js (for certificate display)

## How to Use

1. Fill out the form with your personal information
2. Upload a profile photo
3. Add your education, skills, work experience, and accomplishments
4. Optionally attach certificates or credentials
5. Click "Generate Resume" to preview
6. Download as PDF

## Setup

1. Clone the repository
2. Open `resumegenerator.html` in a web browser
3. No server required - runs locally

## Files

- `resumegenerator.html` - Main form page
- `resumegeneratorfinalresult.html` - Resume preview and download page
- `resumegenerator.js` - JavaScript functionality
- `resumegenerator.css` - Styling for form page
- `resumegeneratorfinalresult.css` - Styling for result page
